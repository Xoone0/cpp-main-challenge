# Day 011

## 今天推进了什么

今天继续推进了 C 语言第三章。

这一阶段的重点已经不再只是“会写输入输出”，
而是开始真正练判断逻辑和分支处理。

今天我主要做了一部分和 if、逻辑运算、switch 有关的练习，
包括：

- 用 if 写两个数差值
- 用 if 求三个数的最小值
- 判断三个数里相等值的情况
- 用逻辑或判断差值范围
- 用 switch 判断奇偶
- 用 switch 判断月份对应的季节

和前两章相比，
第三章明显更考验逻辑拆分能力。

## 今天卡住了什么

今天最大的难点不是语法本身，
而是题目一复杂，就很容易在“条件怎么拆”这一步卡住。

我能明显感觉到：

- 有些题看起来懂了，但自己从头写时还是会卡
- 分支一多，就容易漏情况
- switch 和 if 的写法虽然不长，但逻辑必须先想清楚
- 有几题还是需要先看思路，再回来自己整理

这说明我现在真正要练的，
不是“记住语法”，
而是“怎么把判断逻辑拆出来”。

## 今天最重要的理解变化

我今天更清楚了一点：

第三章难，不是因为 if 和 switch 本身难，
而是因为它开始要求我从“顺着写”切换到“分情况处理”。

这一步一旦能慢慢适应，
后面循环和函数才会更好理解。
