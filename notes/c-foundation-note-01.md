# C Foundation Note 01

## 当前阶段说明

虽然这条主线最终会进入 C++，
但我当前的实际学习路径是：

**先学 C，再学 C++。**

所以现阶段的重点，不是直接跳进更复杂的 C++ 语法和特性，
而是先把更基础的内容打稳。

---

## 为什么先学 C

我现在选择先学 C，主要是因为：

1. C 的基础语法更直接
2. 更容易先建立对程序结构的基本理解
3. 有助于后面理解 C++ 中更复杂的内容
4. 能让我先把“输入、输出、变量、判断、循环、函数”等基础部分走扎实

对我来说，
这不是绕路，
而是在给后面的 C++ 学习铺底。

---

## 当前阶段更重要的事

这个阶段，我更重视这些基础问题：

- 程序最基本的结构是什么
- 变量和数据类型怎么理解
- 输入输出怎么写
- 条件判断和循环怎么用
- 函数是什么，为什么要拆函数
- 写代码时最容易犯哪些基础错误

我不追求一开始就学很多，
我更想先把这些基础打稳。

---

## 当前阶段的记录重点

在这个阶段，我会优先记录：

- C 语言基础练习
- 简单代码片段
- 每天遇到的卡点
- 对基础语法的理解变化
- 进入 C++ 前需要先掌握的内容

---

## 当前阶段一句话总结

我现在正在用 C 语言打基础，
为后续正式进入 C++ 做准备。

先把底打稳，
再往上走。
